# CSS Class İsimleri Referansı

## Önemli Class İsimleri

### Layout Classes
- `.page-wrapper` - Ana sayfa wrapper
- `.main-wrapper` - Ana içerik wrapper
- `.container-large` - Büyük container
- `.container-medium` - Orta container
- `.container-small` - Küçük container
- `.container-xlarge` - Çok büyük container
- `.padding-global` - Global padding
- `.padding-section-large` - Section için büyük padding

### Component Classes
- `.home_workforce-feature_component` - Workforce feature component
- `.home_workforce-features_card` - Workforce feature card
- `.card-1`, `.card-2`, vb. - Kart numaraları
- `.home_hero-header` - Hero section
- `.home_footer-cta_component` - Footer CTA component

### Grid & Layout
- `.w-layout-grid` - Webflow grid
- `.w-dyn-list` - Dynamic list
- `.w-dyn-items` - Dynamic items

### Spacing Classes
- `.margin-bottom` - Alt margin
- `.margin-top` - Üst margin
- `.padding-0` - Padding yok
- `.spacing-clean` - Temiz spacing

### Text Classes
- `.text-size-medium` - Orta text boyutu
- `.text-weight-semibold` - Yarı kalın
- `.text-color-alternate` - Alternatif renk
- `.heading-style-h1` - H1 başlık stili
- `.heading-style-h2` - H2 başlık stili

### Button Classes
- `.button` - Buton base
- `.button.is-small` - Küçük buton
- `.button.is-alternate` - Alternatif buton
- `.button-group` - Buton grubu

### Utility Classes
- `.hide` - Gizle
- `.hide-mobile` - Mobilde gizle
- `.hide-tablet` - Tablette gizle
- `.is-center` - Ortala
- `.is-relative` - Relative positioning

---

## Class İsimlendirme Pattern'i

### BEM Benzeri Yapı
- Component: `home_workforce-feature_component`
- Element: `home_workforce-features_card`
- Modifier: `card-1`, `card-2`

### Prefix Pattern
- `home_` - Homepage'e özel
- `section_` - Section'a özel
- `layout` - Layout component'leri

---

## Notlar

- Webflow class isimleri kullanılıyor
- BEM benzeri ama tam BEM değil
- Utility class'lar yaygın
- Responsive class'lar var (hide-mobile, vb.)
