# Relevance AI - Border Radius & Shadows

## Border Radius
- **Butonlar:** Yuvarlatılmış köşeler (8-12px)
- **Kartlar:** Hafif yuvarlatılmış (8-16px)
- **Navigation Bar:** Hafif yuvarlatılmış üst köşeler
- **Genel:** Modern, yumuşak görünüm

## Shadows
- **Navigation Bar:** Hafif gölge (sayfadan ayrılması için)
- **Kartlar:** Minimal gölge (derinlik için)
- **Butonlar:** Hover'da hafif gölge artışı

## Gradient Kullanımı
- **Butonlar:** Mor gradient (açık → koyu)
- **Vurgular:** Gradient ile dikkat çekme
- **Arka Planlar:** Bazı section'larda gradient kullanımı

---

## Genel Tasarım Notları
- **Minimal, temiz görünüm**
- **Modern UI patterns**
- **Yumuşak geçişler (border radius, gradients)**
- **Hafif gölgeler (derinlik için)**
