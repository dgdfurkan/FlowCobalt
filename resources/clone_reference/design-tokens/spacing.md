# Relevance AI - Spacing Sistemi

## Genel Spacing Prensipleri

### Hero Section
- **Üst Boşluk:** Geniş (padding-top büyük)
- **Alt Boşluk:** Geniş (padding-bottom büyük)
- **İçerik Arası:** Orta-yüksek spacing

### Section'lar Arası
- **Boşluk:** Yüksek (her section arasında geniş boşluk)
- **Amaç:** İçeriğin nefes alması, temiz görünüm

### Component İçi Spacing
- **Kartlar:** Orta padding (16-24px)
- **Butonlar:** Yeterli padding (12-16px yatay, 10-14px dikey)
- **Grid İçi:** Tutarlı gap (24-32px)

---

## Spacing Notları
- **Beyaz alan kullanımı:** Yüksek (minimal, temiz görünüm)
- **Yoğunluk:** Düşük (her element nefes alıyor)
- **Responsive:** Mobilde spacing azalıyor ama oran korunuyor
