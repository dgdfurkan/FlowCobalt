# Relevance AI - Renk Paleti

## Ana Renkler

### Primary (Mor/Purple Gradient)
- **Kullanım:** CTA butonları, vurgular, linkler
- **Gradient:** Açık mor → Koyu mor
- **Örnekler:**
  - "Try for free" butonu
  - "Talk to Sales" butonu
  - "Sign Up" butonu
  - Navigation link hover durumları

### Background
- **Ana Arka Plan:** Beyaz (#FFFFFF)
- **Top Banner:** Koyu mor/lacivert (gradient)
- **Section Arka Planları:** Beyaz, açık gri tonları

### Text
- **Ana Başlık:** Koyu mavi/siyah
- **Alt Başlık:** Koyu gri
- **Body Text:** Koyu gri
- **Vurgu Metin:** Mor (AI agents, human-quality work gibi)

### Butonlar
- **Primary CTA:** Mor gradient arka plan + beyaz text
- **Secondary CTA:** Beyaz arka plan + mor border + koyu text

---

## Renk Notları
- Mor/purple ana marka rengi
- Gradient kullanımı yaygın
- Yüksek kontrast (okunabilirlik öncelikli)
- Modern, teknoloji odaklı görünüm
