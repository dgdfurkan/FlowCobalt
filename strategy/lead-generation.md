# Lead Bulma Stratejisi

## Problem Sinyalleri (Lead Filtreleri)

### 1. İş İlanı Sinyalleri
**"Hiring: Operations Coordinator / Inventory Specialist / Dispatch"** ilanı var
- **Sinyal:** Manuel iş yükü artıyor
- **Aksiyon:** İlgili kişiye direkt ulaş

### 2. Büyüme Sinyalleri
**Shopify/WooCommerce mağazası büyümüş** (çok ürün / çok sipariş)
- **Sinyal:** Süreç tıkanıyor
- **Aksiyon:** Operations Manager'a ulaş

### 3. Mevcut Otomasyon Karmaşası
**"We use Zapier/Make"** diyor ama ekip küçük
- **Sinyal:** Maliyet/karmaşa problemi
- **Aksiyon:** Daha basit çözüm öner

### 4. Sürekli Şikayetler
**"Support backlog / late deliveries"** şikâyeti
- **Sinyal:** Süreç problemi var
- **Aksiyon:** Çözüm odaklı yaklaş

## Lead Scoring

### Yüksek Öncelik (Hemen İletişim)
- İş ilanı var + büyüme sinyali
- Sürekli şikayet var + küçük ekip
- Mevcut otomasyon karmaşası + maliyet problemi

### Orta Öncelik (1-2 Hafta İçinde)
- Büyüme sinyali var
- İş ilanı var

### Düşük Öncelik (Aylık Takip)
- Sadece büyüme potansiyeli var
